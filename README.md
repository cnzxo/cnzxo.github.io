## HUSTOJ在线测评系统
* 下载安装包
~~~
wget https://cnzxo.github.io/sh/hustoj.sh
~~~
* 启动安装程序
~~~
sudo bash hustoj.sh
~~~

## linux tree安装教程
* 下载地址
~~~
wget https://cnzxo.github.io/tgz/tree-1.7.0.tgz
~~~
* 文件解压
~~~
tar -zxvf tree-1.7.0.tgz
~~~
* 安装tree
~~~
make install
~~~
